SAIREN COLOR ARCHIVE — Space×Math×Geometry Theme (Drop-in)

目的:
- 既存の「マトリックス背景」を、宇宙×数学×図形×アーティスティックに置き換えます。
- 既存構成（HOME/WORKS/ABOUT…）は崩しません。

このZIPに入っているのは:
- assets/js/matrix.js  (差し替え用)

適用方法:
1) リポジトリ（公開ルート）の同じパスに上書きコピーしてください:
   /assets/js/matrix.js

2) すでに各ページで /assets/js/matrix.js を読み込んでいる前提です。
   (読み込んでいない場合は、各HTMLの <head> か </body> 直前に
    <script src="/assets/js/matrix.js" defer></script>
    を追加してください)

確認ポイント:
- 背景が「宇宙グラデ + 数式記号の雨 + 図形オーバーレイ」になれば成功
- ページ遷移クリック時に一瞬ブーストする
- 見出し/カード文が短時間だけ崩れて収束する（長文は崩しません）
