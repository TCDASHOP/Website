# SAIREN COLOR ARCHIVE

A lightweight multi-device (mobile / tablet / PC) single-design website.
All devices share the same UI/UX (no desktop-only navigation).

## Structure

- `index.html` : Home
- `works.html` : Works (formed/main only)
- `about.html` : About
- `sns.html` : SNS links (logo icons)
- `contact.html` : Contact
- `license.html` : License

### Assets
- `assets/site.config.json` : site settings + i18n + works data
- `assets/css/style.css` : styles
- `assets/js/` : scripts

## Works images

Use **formed/main only** to keep the site fast.

Recommended:
- `assets/images/works/formed/` : used
- `assets/images/works/raw/` : optional (can be removed if not used)

## SNS icons

Place logos here:
- `assets/images/social/Instagram-logo.png`
- `assets/images/social/TikTok-logo.png`

## Deploy

This repo is designed for static hosting (e.g. GitHub Pages / Cloudflare Pages).
