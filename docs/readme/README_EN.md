# SAIREN COLOR ARCHIVE (English)

This repository hosts the static website for **sairencolorarchive.com** via GitHub Pages.

## Language folders
- Japanese: `/` (root)
- English: `/en/`
- Spanish: `/es/`
- French: `/fr/`
- Korean: `/ko/`
- Chinese (Simplified): `/zh-hans/`

## Important files
- Root pages: `index.html`, `works.html`, `about.html`, `sns.html`, `contact.html`, `license.html`, `lookbook.html`
- Assets: `/assets/`
- SEO: `/robots.txt`, `/sitemap.xml`

## Update (quick)
1. GitHub → Add file → Upload files
2. Commit
3. Wait for GitHub Pages deploy
4. Verify on https://sairencolorarchive.com/

## Design policy
- Keep the **same design and experience** across mobile/tablet/desktop.
