# SAIREN COLOR ARCHIVE (한국어)

이 저장소는 GitHub Pages를 통해 **sairencolorarchive.com** 정적 웹사이트를 호스팅합니다.

## 언어 폴더
- 일본어: `/` (루트)
- 영어: `/en/`
- 스페인어: `/es/`
- 프랑스어: `/fr/`
- 한국어: `/ko/`
- 중국어(간체): `/zh-hans/`

## 주요 파일
- 루트 페이지: `index.html`, `works.html`, `about.html`, `sns.html`, `contact.html`, `license.html`, `lookbook.html`
- 에셋: `/assets/`
- SEO: `/robots.txt`, `/sitemap.xml`

## 업데이트(간단)
1. GitHub → Add file → Upload files
2. Commit
3. GitHub Pages 배포 완료 후
4. https://sairencolorarchive.com/ 에서 확인

## 디자인 원칙
- 모바일/태블릿/데스크톱에서 **동일한 디자인과 경험**을 유지합니다.
