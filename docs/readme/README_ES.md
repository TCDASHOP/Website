# SAIREN COLOR ARCHIVE (Español)

Este repositorio aloja el sitio estático de **sairencolorarchive.com** mediante GitHub Pages.

## Carpetas de idioma
- Japonés: `/` (raíz)
- Inglés: `/en/`
- Español: `/es/`
- Francés: `/fr/`
- Coreano: `/ko/`
- Chino (simplificado): `/zh-hans/`

## Archivos importantes
- Páginas en la raíz: `index.html`, `works.html`, `about.html`, `sns.html`, `contact.html`, `license.html`, `lookbook.html`
- Recursos: `/assets/`
- SEO: `/robots.txt`, `/sitemap.xml`

## Actualización (rápida)
1. GitHub → Add file → Upload files
2. Commit
3. Esperar el despliegue de GitHub Pages
4. Verificar en https://sairencolorarchive.com/
