# SAIREN COLOR ARCHIVE（简体中文）

此仓库用于通过 GitHub Pages 托管静态网站 **sairencolorarchive.com**。

## 语言目录
- 日语：`/`（根目录）
- 英语：`/en/`
- 西班牙语：`/es/`
- 法语：`/fr/`
- 韩语：`/ko/`
- 简体中文：`/zh-hans/`

## 重要文件
- 根目录页面：`index.html`, `works.html`, `about.html`, `sns.html`, `contact.html`, `license.html`, `lookbook.html`
- 资源：`/assets/`
- SEO：`/robots.txt`, `/sitemap.xml`

## 更新（快速）
1. GitHub → Add file → Upload files
2. Commit
3. 等待 GitHub Pages 部署完成
4. 访问 https://sairencolorarchive.com/ 检查

## 设计原则
- 在手机/平板/桌面端保持**相同的设计与体验**。
